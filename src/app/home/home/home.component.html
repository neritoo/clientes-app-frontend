Aplicación de clientes

En esta aplicación logramos realizar un CRUD completo de clientes a una base de datos, utilizando tecnologías de punta para el lado del servidor, así como para el lado del cliente.

Las tecnologías utilizadas son:

Angular Spring Framework 5 MySql

Para el lado del cliente se utilizó la tecnología de Angular, un framework que permite trabajar de manera modular con JavaScript, HTML y CSS.

Para el lado del servidor se usó Java, con el poderoso framework para desarrollo web de Spring Boot. Se manejan el modelo y los controladores correspondientes.

Como base de datos se utilizó MySql, una base de datos relacional sencilla de usar.
